그림보고 랜덤 빙고 게임
선생님이 그림을 보여주면 학생이 빙고판에서 같은 단어를 찾아 클릭하는 게임.

🎒 학생: 빙고판의 단어 중 같은 것 클릭

문제읽고 랜덤 빙고 게임
엑셀로 만든 문제/정답 세트를 업로드. 학생은 문제를 읽고 빙고판에서 정답 그림을 클릭.
🧑‍🏫 선생님: 엑셀 업로드 → 문제 클릭 →
🎒 학생: 빙고판의 그림 중 정답 클릭
학생 보드 = 그림 출제 = 문제(텍스트) 엑셀 업로드
