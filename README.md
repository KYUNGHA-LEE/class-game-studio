# Class Game Studio — 실시간 수업용 빙고 게임 모음

선생님과 학생이 한 화면에서 함께 즐기는 실시간 멀티플레이어 수업 도구입니다. 별도의 설치 없이 웹 브라우저만으로 빙고 형식의 참여형 활동을 진행할 수 있습니다.

## 주요 기능

- **그림 보고 랜덤 빙고 게임** — 선생님이 그림을 제시하면, 학생은 자신의 빙고판에서 같은 의미의 단어를 찾아 클릭합니다.
- **문제 읽고 랜덤 빙고 게임** — 엑셀(`.xlsx`, `.xls`, `.csv`)로 만든 문제·정답 세트를 업로드하면, 학생이 문제를 읽고 빙고판의 정답 그림을 클릭합니다.
- **실시간 동기화** — Firebase Realtime Database로 선생님의 출제와 학생의 응답이 즉시 반영됩니다.
- **반별 분리 진행** — 같은 주소에 `?room=2학년3반`처럼 방 이름을 붙이면 여러 반이 동시에 따로 진행할 수 있습니다.
- **엑셀 문제 업로드** — 별도 입력 화면 없이 표 형태의 문제 세트를 그대로 불러옵니다.

## 실행 방법

이 프로젝트는 빌드 과정이 없는 정적 웹 사이트로, 모든 코드가 HTML 파일 안에 포함되어 있습니다.

### 방법 1 — 배포된 사이트 사용
Firebase Hosting에 배포된 주소(`https://bing-go-game.web.app`)에 접속해 바로 사용합니다.

### 방법 2 — 로컬에서 확인
Firebase 인증과 실시간 DB 연동이 필요하므로, 파일을 더블클릭하는 대신 로컬 서버로 실행하는 것을 권장합니다.

```bash
# Firebase CLI 설치 후
firebase serve
```

또는 동봉된 `배포하기.bat`(Windows)을 사용해 배포할 수 있으며, 자세한 절차는 `배포_가이드.md`에 단계별로 정리되어 있습니다.

```bash
firebase deploy
```

## 기술 스택

- HTML · CSS · 바닐라 JavaScript (프레임워크·번들러 없이 단일 페이지로 구성)
- Firebase Realtime Database / Storage / Hosting (프로젝트: `bing-go-game`)
- SheetJS (xlsx) — 엑셀 문제 세트 파싱
- Noto Sans KR 웹폰트
